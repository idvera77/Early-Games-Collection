# Early Games & Academic Projects
Este repositorio es una recopilación de mis proyectos de desarrollo de videojuegos realizados durante mi **Grado Superior en Desarrollo de Aplicaciones Multiplataforma (DAM)**. 

Estos proyectos demuestran mi capacidad para integrar lógica de juego con **persistencia de datos en bases de datos relacionales**, una habilidad fundamental en el desarrollo de software profesional.

---
## Proyectos Incluidos

### 1. Popollo Adventures (Java & Android)
Un juego de rol desarrollado en dos versiones. Este proyecto me permitió entender la portabilidad de código entre Java estándar y el ecosistema Android.
* **Tecnologías:** Java, Android SDK, MySQL, SQLite.
* **Base de Datos:** Uso de SQLite para la gestión local de personajes, inventarios y estadísticas del jugador.

### 2. Once Upon A Chicken (Android)
Un juego de plataformas sencillo diseñado específicamente para poner a prueba el motor LibGDX.
* **Tecnologías:** Java, LibGDX, SQLite.
* **Habilidades:** Ciclo de vida del juego (Game Loop), manejo de físicas simples y optimización para Android.

### 3. Visual Novel (Proyecto Final de Grado)
Mi proyecto final de ciclo. Una novela visual que combina narrativa interactiva con una arquitectura de software más robusta.
* **Tecnologías:** Java, MySQL, SQLite.
* **Base de Datos:** Gestión de flujos narrativos y persistencia de decisiones del usuario mediante base de datos.

---
## Stack Tecnológico
* **Lenguajes:** Java.
* **Bases de Datos:** SQLite, MySQL (Modelado en MySQL Workbench).
* **Frameworks:** LibGDX, Android SDK.
* **Herramientas:** Android Studio / NetBeans.
